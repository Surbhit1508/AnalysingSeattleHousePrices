# Seattle Housing Market Analysis Dashboard (1990-2015)

## Overview
This Power BI dashboard provides comprehensive insights into Seattle's housing market trends and characteristics over a 25-year period (1990-2015). The dashboard visualizes key metrics including price distributions, property features, and historical trends to support real estate analysis and decision-making.

## Data Sources
- Historical house prices in Seattle from 1990 to 2015
- Property characteristics including:
  - Number of bedrooms and bathrooms
  - House condition ratings
  - Construction year (built date)
  - Number of floors
  - Property grade ratings

## Dashboard Components

### Key Metrics
- Average House Price: $540.09K (latest)
- Average Number of Bedrooms: 3.37
- Average Grade Rating: 7.66 out of 13

### Visualizations

1. **Condition of House**
   - Bar chart showing distribution of house conditions
   - Scale ranges from 1-5
   - Includes count of properties in each condition category

2. **Average Price by Number of Floors**
   - Horizontal bar chart
   - Shows price correlation with property height
   - Floors range from 1-4 stories

3. **Average Price by Bathrooms**
   - Line chart showing price trends relative to bathroom count
   - Ranges from 0-8 bathrooms
   - Demonstrates price scaling with bathroom quantity

4. **Average Price by Ratings**
   - Bar chart showing price distribution across different rating categories
   - Ratings range from 0-4
   - Shows correlation between property rating and market value

5. **Historical Price Trends**
   - Line graph tracking average prices by construction year
   - Spans from 1900 to 2000+
   - Highlights key periods in Seattle's real estate development

6. **Grade Rating Distribution**
   - Gauge chart showing average property grade
   - Scale of 1-13
   - Current average: 7.66

## Usage Guide

### Filtering Options
- Users can filter data by:
  - Construction year
  - Number of bedrooms
  - Number of bathrooms
  - Property condition
  - Grade rating

### Interactive Features
- Hovering over charts reveals detailed tooltips
- Cross-filtering enabled between visualizations
- Drill-down capabilities for detailed analysis
